# Word Analytics Tool

A high-performance, real-time text analysis application built with **Vanilla JavaScript**. This tool helps users monitor their writing metrics instantly, with a focus on social media constraints.

---

## Live Demo
[View the Live Site](https://kanidehbokri.github.io/word-analytics-tool/)

---

## Features
* **Real-time Counting:** Instantly tracks words and characters as you type.
* **Social Media Validation:** Visual alerts for Twitter (X) and Facebook character limits.
* **Clean UI:** Minimalist and responsive design for a distraction-free experience.
* **No Dependencies:** Built with pure JavaScript for maximum performance.

---

## Built With
* **HTML5** - For structured content.
* **CSS3** - For modern, responsive styling.
* **JavaScript (ES6+)** - For real-time logic and DOM manipulation.
